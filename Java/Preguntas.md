1. ¿Qué es Java? Java es un lenguaje de programación de alto nivel desarrollado por Sun Microsystems en 1995.
    
2. ¿Cuáles son las características principales de Java? Las características principales de Java son la portabilidad, seguridad y orientación a objetos.
    
3. ¿Qué es un objeto en Java? Un objeto en Java es una instancia de una clase y tiene sus propios métodos y variables.
    
4. ¿En qué se diferencia una clase de un objeto en Java? Una clase es un modelo que define el comportamiento de un objeto, mientras que un objeto es una instancia de una clase.
    
5. ¿Qué es una interfaz en Java? Una interfaz en Java es una colección de métodos abstractos y constantes. Es utilizada para definir los comportamientos esperados por una clase.
    
6. ¿Qué son los paquetes en Java? Los paquetes en Java son una forma de organizar clases y interfaces en un mismo espacio de nombres.
    
7. ¿Qué es la herencia en Java? La herencia en Java es un concepto que permite a una clase heredar atributos y métodos de otra clase.
    
8. ¿Qué es el polimorfismo en Java? El polimorfismo en Java es la capacidad de los objetos de una clase para tomar varias formas.
    
9. ¿Cómo maneja Java la excepciones? Java maneja las excepciones utilizando bloques try-catch-finally.
    
10. ¿Cuál es la diferencia entre JDK y JRE en Java? JDK (Java Development Kit) es un kit de desarrollo que proporciona herramientas para desarrollar código Java, mientras que JRE (Java Runtime Environment) es un entorno de tiempo de ejecución que permite la ejecución de código Java en una máquina.
